# Section 2: Hero Section

> **Version:** v2.9.0 | **Priority:** Phase 1

## Web Terms
- **Hero Section:** First prominent visual area containing headline, subheadline, CTA
- **Above the Fold:** Visible without scrolling
- **CTA (Call-to-Action):** Primary button prompting action

## Purpose
Immediately communicate who you are and what you do. Provide clear next action.

## Structure

```
┌─────────────────────────────────────────┐
│                                         │
│      Taeho (TK) Kim                     │  ← Name
│      Fullstack UX Engineer              │  ← Title
│                                         │
│      Building intuitive products        │  ← Value proposition
│      and empowering engineering teams   │
│                                         │
│        [ Download Resume ➜ ]            │  ← Primary CTA
│                                         │
│              ↓ scroll                   │  ← Scroll indicator
└─────────────────────────────────────────┘
```

## Content Options

**Headline (pick one):**
- "Building Products That Scale"
- "Code. Design. Lead."
- "Where Engineering Meets User Experience"

**Subheadline:**
> 20+ years crafting web applications with React, Python, and Django.
> Passionate about intuitive products and growing engineering teams.

## CTA
- **Primary:** "Download Resume" (links to PDF)
- **Secondary (optional):** "View Projects" (scrolls to portfolio)

## Implementation
- Use existing gradient background
- Add Framer Motion entrance animation
- Animated scroll indicator (bouncing chevron)
